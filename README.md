# HKL - Hollow Knight Loader

A mod manager for Hollow Knight focused on user experience.

## Features

- Install and manage mods from ModLinks
- Create mod packs for quick preset switching
- Automatic Modding API installation
- Update all mods with one click
- Clean, minimal interface

## Installation

Download the latest release from the [Releases](../../releases) page:
- **HKL.exe** - Portable executable (no installation required)
- **HKL-Setup.exe** - Installer with desktop shortcuts

## Usage

1. Launch HKL and set your Hollow Knight game directory
2. Browse available mods in the Mods tab
3. Click install on any mod to download and enable it
4. Create mod packs in the Packs tab for quick preset switching

## Compatibility Note

For maximum platform compatibility, consider using established mod managers like [Lumafly](https://github.com/TheMulhima/Lumafly) or [Scarab](https://github.com/fifty-six/Scarab). HKL is an opiniated alternative focused on my ideal mod manager.

## Building from Source

```bash
npm install
npm run build
```
